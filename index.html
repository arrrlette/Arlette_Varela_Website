    World Happiness Report:

    World Happiness Report
    Our main objective was to explore the measures of subjective well-being as laid out in the World Happiness Report to determine which ones had the most influence on happiness. Python, Jupyter Notebooks, Pandas, and Matplotlib were utilized in this project.

    Home Price Prediction Model:

    Home Price Prediction Model
    We built a machine learning model and web tool that takes in user input and predicts townhome and single-family home prices in Apex, NC based on square-footage, bedrooms, bathrooms, and a few other things that make a house unique. Python, Jupyter Notebooks, Pandas, TensorFlow, HTML/CSS, JavaScript, and Tableau were utilized in this project.

    SuperHero Database:

    SuperHero Database
    For this project, we created a website that displays superhero and villain information from multiple universes. We also created a battle section that allows you to pick between two fighters and see who would win based on their stats. We accessed the superhero database api listed above and loaded the json data into a mongo database. HTML/CSS, Javascript, Python, Flask, MongoDB and PyMongo were utilized in this project.
